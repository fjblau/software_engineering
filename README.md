# Software Engineering Bootcamp
**Digital Campus Vorarlberg**

---

## Bootcamp Overview

A comprehensive 16-week software engineering bootcamp covering full-stack development, professional tools, and modern programming practices.

---

## Course Structure

### **Week 1: Professional Development Environment**
- Command Line Interface (CLI) fundamentals
- Git & GitHub version control
- VS Code setup and shortcuts
- Professional workflow basics

### **Week 2: Programming Fundamentals**
- JavaScript & Python fundamentals
- Variables, data types, and operators
- Control flow (conditionals & loops)
- Functions and scope
- Basic testing and debugging

### **Week 3: Data Structures & Algorithms Fundamentals**
- Arrays and lists
- Sorting algorithms
- Time and space complexity
- SQL basics and queries
- Python data structures

### **Week 4: Intermediate Programming**
- Object-oriented programming basics
- Design patterns introduction
- Advanced SQL queries
- Python classes and methods
- JavaScript objects and prototypes

### **Week 5: Web Fundamentals**
- HTML5 semantic markup
- CSS3 styling and layouts
- JavaScript DOM manipulation
- HTML forms and validation
- Responsive design basics

### **Week 6: Frontend Fundamentals**
- CSS advanced techniques
- JavaScript events and listeners
- Interactive UI components
- Form handling
- Browser APIs

### **Week 7: Databases - SQLite & Relational Concepts**
- SQLite database design
- Relational database theory
- SQL CRUD operations
- Joins and complex queries
- Data normalization

### **Week 8: Backend Architecture Basics**
- Server-side concepts
- RESTful API principles
- Backend frameworks introduction
- Request/response cycles
- Middleware concepts

### **Week 9: Backend Development with Node.js & Python**
- Express.js framework
- Flask/FastAPI basics
- Route handling
- Request/response processing
- Error handling

### **Week 10: CI/CD & GitHub Actions**
- Continuous Integration/Continuous Deployment
- GitHub Actions workflows
- Automated testing pipelines
- Build automation
- Deployment strategies

### **Week 11: Frontend Frameworks - React & TypeScript**
- React fundamentals
- JSX syntax and components
- TypeScript introduction
- Type safety in JavaScript
- React hooks and state management

### **Week 12: Advanced Data & Queries**
- Complex SQL patterns
- JavaScript advanced concepts
- Python data processing
- Query optimization
- SQL performance tuning

### **Week 13: Professional Development Practices**
- Code reviews and collaboration
- Git workflow best practices
- Professional documentation
- Team communication
- Agile methodologies

### **Week 14: Advanced Databases**
- PostgreSQL relational database
- MongoDB document database
- Redis caching and sessions
- Neo4j graph database
- Database selection and architecture

### **Week 15: DevOps & Deployment**
- Docker containerization
- Container orchestration
- Docker Compose
- Monitoring and logging
- TypeScript advanced patterns
- Production deployment

### **Week 16: Professional Capstone Project - Part 1**
- Full-stack application development
- Microservices architecture
- Dual-language implementation (JavaScript + Python)
- Professional project planning
- Sprint-based development
- Testing and quality assurance

## Getting Started

### Prerequisites
- Computer with macOS, Windows, or Linux
- Internet connection
- Willingness to learn!

### Setup Instructions

1. **Install Git**
   ```bash
   # macOS
   brew install git
   
   # Windows
   # Download from https://git-scm.com/
   
   # Linux
   sudo apt install git
   ```

2. **Install Node.js**
   ```bash
   # macOS
   brew install node
   
   # Windows/Linux
   # Download from https://nodejs.org/
   ```

3. **Install Python**
   ```bash
   # macOS
   brew install python3
   
   # Windows/Linux
   # Download from https://python.org/
   ```

4. **Install VS Code**
   - Download from https://code.visualstudio.com/

5. **Clone This Repository**
   ```bash
   git clone [repository-url]
   cd software_engineering
   ```

## Weekly Format

Each week follows a structured format:
- **Daily Standups**: Morning and afternoon check-ins

python3 --version

docker --version && docker-compose --version

docker exec -it postgres-db psql -U postgres
